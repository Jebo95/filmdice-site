# FilmDice site (privacy + support for App Store)

Three static pages, no dependencies:
- `index.html` — minimal landing
- `privacy.html` — Privacy Policy (use as the **Privacy Policy URL**)
- `support.html` — Support (use as the **Support URL**)

## Deploy on GitHub Pages (free)
1. Create a public repo, e.g. `filmdice-site`.
2. Put these files in the repo root and push.
3. Repo → Settings → Pages → Source: "Deploy from a branch", branch `main`, folder `/ (root)`.
4. Your URLs (after ~1 min):
   - Privacy: `https://<user>.github.io/filmdice-site/privacy.html`
   - Support: `https://<user>.github.io/filmdice-site/support.html`

Paste those into App Store Connect. Must be live before submitting for review.
